People and tools behind Lunex

Lunex is a solo project by David Dev, so the site should say that plainly. The work is serious, but it still has the energy of something being built by hand.

David Dev

The language, the direction, the release flow, and most of the day-to-day decisions come from one person doing the work directly.

Zig

Zig deserves real credit here. It makes the runtime side feel practical, low-level, and portable in a way that fits Lunex very well.

Go

The compiler side and the surrounding tooling benefit from the Go ecosystem and its straightforward build story.

The users

Anyone testing, reporting issues, or writing examples helps the project move forward, even when things are still rough around the edges.
