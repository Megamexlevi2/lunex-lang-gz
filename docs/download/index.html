Download NTL

These are the official release assets from the GitHub releases folder. Pick the one that matches the machine you are on, and keep going.

The release directory is the same one you gave: https://github.com/Megamexlevi2/ntl-lang-gz/releases/latest/download.

Quick install examples

Linux / macOS

Download the file, make it executable if needed, and place it somewhere on your PATH.

curl -LO https://github.com/Megamexlevi2/ntl-lang-gz/releases/latest/download/ntl-linux-amd64
chmod +x ntl-linux-amd64
sudo mv ntl-linux-amd64 /usr/local/bin/ntl

Windows

Use the installer or place the executable in a folder that is already on your PATH.

powershell -ExecutionPolicy Bypass -File install.ps1
