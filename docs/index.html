Lunex GZ

Lunex officially started on Termux before expanding into Linux and other desktop platforms. That origin still influences the project today, especially in the way the runtime is designed around portability, low-level control, and practical tooling that works well even in lightweight environments. Linux support is now mature and stable, while Windows support has reached a solid and usable state, though some parts of the ecosystem are still being refined as the platform layer continues to evolve.
