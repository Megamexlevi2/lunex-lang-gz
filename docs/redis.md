# Redis Module

Redis client for caching, data structures, and pub/sub operations.

**Use case:** Use Redis for caching, sessions, and real-time data.

---

## Import

```ntl
val redis = @import("std.redis")
```

---

## Available Functions

**Signature:**
